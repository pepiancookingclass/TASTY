# **App Name**: Tasty

## Core Features:

- Chef Profile Creation: Allow chefs to create profiles, upload photos, and manage their offerings (pastries, desserts, savory items) with prices and descriptions in both English and Spanish.
- User Profile Management: Enable users to create profiles with basic information, including phone number, email, profile picture, and delivery address with geolocation support.
- Product Listing and Display: Display available pastries, desserts, and savory items with photos, descriptions, and prices. Allows for filtering based on dietary restrictions or ingredients using a tool that reviews listings and generates labels.
- Order Placement and Cart System: Implement a shopping cart system for users to add items, specify delivery address, and view order summaries, showing individual price, delivery price and platform fee of 10%.
- Delivery Time Management: Enable chefs to set the required advance notice for orders, displayed as a warning to the user during order placement.
- Promotions and Offers: Create a section for displaying seasonal promotions and special offers (e.g., discounts, free items with purchase). Offer creation form allows admin to specify offer conditions and connect it to discount system.
- Multi-Role Management: Implement role-based access control for administrators, service agents, chefs, and users, with different permissions for each role.

## Style Guidelines:

- Primary color: A hot pink (#FF1493) to evoke a sense of fun, energy, and the homemade quality of the products.
- Background color: A very light and airy pink (#FFE4E1) to keep the design gentle and inviting.
- Accent color: A vivid, almost neon orange (#FF4500) to draw attention to key interactive elements and promotions.
- Headline font: 'Belleza', chosen for its humanist sans-serif qualities that align with fashion, art, and design. Body text: 'Alegreya' to ensure clear and sophisticated readability for descriptions and specifications.
- Employ hand-drawn style icons for categories, dietary information, and user actions to enhance the homemade theme and add a personal touch.
- Use a clean, spacious layout featuring high-quality product photos to emphasize visual appeal and build trustworthiness.
- Incorporate subtle animations, such as smooth transitions and hover effects, to enhance user engagement and provide feedback without being distracting.
